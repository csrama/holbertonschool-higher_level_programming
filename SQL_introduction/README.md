SQL - Introduction
Description
This project is an introduction to SQL and MySQL.
It focuses on understanding relational databases and performing basic SQL operations including database creation, table management, and data manipulation.

All tasks are executed using MySQL 8.0 on Ubuntu 22.04 LTS.

Learning Objectives
At the end of this project, you should be able to explain:

What a database is

What a relational database is

What SQL stands for

What MySQL is

How to create and delete a database

The difference between DDL and DML

How to CREATE and ALTER tables

How to SELECT data from tables

How to INSERT, UPDATE, and DELETE data

How to use aggregate functions (COUNT, AVG, etc.)

How to filter results using WHERE

How to group results using GROUP BY

How to sort results using ORDER BY

Requirements
Allowed editors: vi, vim, emacs

All files end with a new line

Each SQL file starts with a task description comment

All SQL queries must have a comment before them

All SQL keywords must be written in UPPERCASE

A README.md file is mandatory
