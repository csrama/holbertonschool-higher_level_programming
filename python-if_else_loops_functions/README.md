# Task Descriptions

0. positive_or_negative.py
Description: Determines if a random number is positive, negative, or zero
Output format:

{number} is positive

{number} is negative

{number} is zero

1. last_digit.py
Description: Prints the last digit of a random number and classifies it
Output format:

Last digit of {number} is {last_digit} and is greater than 5

Last digit of {number} is {last_digit} and is 0

Last digit of {number} is {last_digit} and is less than 6 and not 0

2. print_alphabet.py
Description: Prints the ASCII alphabet in lowercase
Output: abcdefghijklmnopqrstuvwxyz
Requirements: One print call, one loop, no string storage

3. print_alphabt.py
Description: Prints the ASCII alphabet in lowercase except 'q' and 'e'
Output: abcdfghijklmnoprstuvwxyz
Requirements: One print call, one loop, no string storage

4. print_hexa.py
Description: Prints numbers 0 to 98 in decimal and hexadecimal
Output format: {decimal} = {hexadecimal}

5. print_comb2.py
Description: Prints numbers from 0 to 99 separated by commas
Output: 00, 01, 02, ..., 98, 99
Requirements: Two digits minimum, last number ends with newline

6. print_comb3.py
Description: Prints all possible different combinations of two digits
Output: 00, 01, 02, ..., 78, 89
Constraints:

Numbers separated by commas

Two digits minimum

01 and 10 are considered the same combination

Print only the smallest combination (01 not 10)

Last number ends with newline

7. islower.py
Description: Function that checks for lowercase character
Prototype: def islower(c):
Returns: True if c is lowercase, False otherwise

8. uppercase.py
Description: Prints a string in uppercase followed by newline
Prototype: def uppercase(str):
Requirements: Use string's format() and ord()/chr()

9. print_last_digit.py
Description: Prints the last digit of a number
Prototype: def print_last_digit(number):
Returns: Value of the last digit

10. add.py
Description: Adds two integers and returns the result
Prototype: def add(a, b):
Returns: Sum of a and b

11. pow.py
Description: Computes a to the power of b and returns the value
Prototype: def pow(a, b):
Returns: a ^ b

12. fizzbuzz.py
Description: Prints numbers from 1 to 100 with FizzBuzz rules
Rules:

Multiples of 3: Fizz

Multiples of 5: Buzz

Multiples of both 3 and 5: FizzBuzz
Output: Numbers and words separated by spaces
